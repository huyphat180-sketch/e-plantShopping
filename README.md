# e-plantShopping

e-plantShopping is a React and Redux based online plant shopping application developed as part of the IBM Front-End Development course.

## Features

* Browse houseplants by category
* Add plants to the shopping cart
* Update item quantities
* Remove items from the cart
* View total cart value
* Redux state management

## Technologies Used

* React
* Redux Toolkit
* React Router
* CSS
